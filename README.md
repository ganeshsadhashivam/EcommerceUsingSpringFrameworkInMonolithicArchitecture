# Ecommerce Using Spring Framework (Monolithic Architecture)

## 📌 Project Overview

This project is a simple Ecommerce Web Application developed using the Spring Framework with Monolithic Architecture.

The application is being built to learn and implement core enterprise application concepts such as:

* Spring MVC
* Dependency Injection (IoC)
* JDBC Connectivity
* Maven
* CRUD Operations
* Layered Architecture
* Database Integration
* Product Management

---

## ▶️ How to Run the Project

### 1. Clone Repository

```bash
git clone https://github.com/ganeshsadhashivam/EcommerceUsingSpringFrameworkInMonolithicArchitecture.git
```

### 2. Import Project

* Open Eclipse
* Import Existing Maven Project

### 3. Configure Database

Update database credentials in:

```text
DBConfig.java
```

### 4. Run on Tomcat Server

* Configure Apache Tomcat 11
* Run project on server
